# gebrauchtwagen-go

Simple public Go prototype repository.

## Structure

- `proto-go/` - minimal runnable Go prototype
- `TASK_DESCRIPTION.md` - task statement
- `SUBMISSION_README.md` - submission notes

## Run

```bash
cd proto-go
./run.sh
```

or

```powershell
cd proto-go
./run.ps1
```

or directly:

```bash
cd proto-go
go run .
```
